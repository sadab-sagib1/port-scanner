# Simple TCP Port Scanner (Python)

A beginner port scanner that checks a small list of common TCP ports on a target.

---

## What it does
- Takes a hostname or IP
- Resolves hostname to IP
- Checks common ports (like 22, 80, 443)
- Prints which ports are open

---

## Run
```bash
python3 port_scanner.py google.com
